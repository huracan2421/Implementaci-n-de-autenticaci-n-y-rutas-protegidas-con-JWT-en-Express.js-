# Implementaci-n-de-autenticaci-n-y-rutas-protegidas-con-JWT-en-Express.js-
se creo una paina web para comprobar las credenciales de forma errónea y correctas 

El servidor se ejecutará en **http://localhost:3000**

## 🧪 Pruebas realizadas

- Login con credenciales correctas e incorrectas
- Protección de rutas mediante middleware
- Validación de token JWT
- Cierre de sesión (eliminación de cookie)
- Acceso denegado sin autenticación
